# 📝 To-Do App

A simple and responsive **To-Do List App** built using **HTML, CSS, and JavaScript**.  
It allows users to add, mark complete, delete, and store tasks locally using the browser’s `localStorage`.

---

## 🚀 Add Features

- ➕ Add new tasks easily  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 💾 Auto-save tasks in localStorage  
- ⚡ Add task with Enter key  
- 📱 Fully responsive & mobile-friendly  
- 🎨 Clean and modern UI with smooth animations

---

## 🛠️ Tech Stack

- **HTML5** – structure  
- **CSS3** – styling and responsive design  
- **JavaScript (ES6)** – functionality & localStorage management
